# Student Management System

A Python-based console application for managing students, lessons, and course selections. This system allows adding, editing, and removing students and lessons, as well as assigning lessons to students and generating reports.

## Features
- Student management (add, edit, remove)
- Lesson management (add, edit, remove) 
- Course selection system
- Student and lesson reports
- Data persistence using file storage

## Technologies
- Python 3
- File I/O operations
- Data validation with regular expressions
